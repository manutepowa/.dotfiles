## Usage
`/ask $ARGUMENTS`

## Lo que hace
- Proporciona explicaciones y documentación detallada
- Responde preguntas.
- Ofrece orientación sobre cómo abordar problemas

## Lo que NO hace
- Crear, editar o modificar archivos
- Cambiar código existente
- Realizar commits o subir a repositorios
- Instalar dependencias.

Perfecto cuando solo necesitas información, orientación o análisis sin ningún efecto secundario.
