# Resolución por IA
  Resumen de Reunión - Sistema ODOO para empresa Mobal

  Funcionalidades ya implementadas:

  - Verificación de documentos con CSV (proporcionado por ellos)
  - Firma de documentos mediante CSV
  - Seguimiento de procesos de verificación y trazabilidad
  - Sistema de Instancias (expone-solicita-archivos)
  - Firma digital que se añade al PDF con código guardado en nodo
  - Gestión de estados para notificaciones de trámites: recibido, en tramitación, resuelto

  Requisitos solicitados por Vicente:

  - Comunicación masiva con documento firmado y validación del csv
  - Comentarios internos exclusivos para administradores en expedientes
  - Asignación de técnicos responsables de expedientes
  - Expedientes internos con sistema de etiquetas y posibles relaciones
  - Sistema de mensajería integrada en expedientes para comunicación constante
  - Notificaciones push (actualmente usan app externa "cutre")

  Estructura organizativa:

  - Cada expediente tiene un técnico asignado
  - Los expedientes internos también requieren asignación de técnicos
  - Posible sistema de socios o expedientes relacionados

  El documento muestra un sistema avanzado de gestión documental con enfásis en la trazabilidad y comunicación
  interna.
